# Third-party components inside the Vulcan Office binary

The Vulcan Office Windows installer bundles the following third-party
components. Each entry lists the component, its license, and a link to the
upstream project. This file is required by AGPL-3.0 sec. 5(a) preservation
of appropriate notices.

## Editor shell (Euro-Office desktop-apps)

- License: **AGPL-3.0-or-later**
- Source: https://github.com/Euro-Office/desktop-apps
- Version pinned: commit `e5f89e272861` (2026-07-01)
- Modifications by Vulcan: branding overlay only, no source patches. See
  `desktop/branding/` for the exact overlay applied.

## Fonts embedded in documents

The editor ships with a set of open fonts to guarantee document rendering
consistency across machines. Each font retains its original license
(SIL Open Font License 1.1 for the majority). Full list is generated by
the upstream build under `Common/fonts/` and shipped verbatim.
