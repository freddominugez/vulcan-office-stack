Corresponding source (AGPL-3.0 sec. 6(d))
--------------------------------------------------------------------------

The complete corresponding source used to produce this binary is publicly
available at:

  https://github.com/freddominugez/vulcan-office-stack

Every release tag (e.g. v9.3.1-vulcan.1) in that repository pins:
  1. The upstream Euro-Office/desktop-apps commit used as the shell source.
  2. The upstream Euro-Office/DesktopEditors tag used as the engine.
  3. The exact branding overlay applied (under desktop/branding/).
  4. The build script that produces this installer (desktop/build/).

To obtain the source used for a specific version:

  git clone --branch <tag> https://github.com/freddominugez/vulcan-office-stack

--------------------------------------------------------------------------
Binary distribution
--------------------------------------------------------------------------

The Windows installer for Vulcan Office is unsigned. Windows SmartScreen
displays "Windows protected your PC" on first execution. To proceed:

  1. Click "More info"
  2. Click "Run anyway"

This behavior stops after the binary earns reputation, or immediately when
a code-signing certificate is added in a future release.

Cryptographic integrity of the download is provided via a detached OpenPGP
signature over sha256sums.txt. The public key for "Vulcan Office Releases
<releases@vulcanoffice.com>" is published inside the installer under
desktop\keys\releases-pubkey.asc and mirrored at
https://vulcanoffice.com/downloads/keys/releases-pubkey.asc
